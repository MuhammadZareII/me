Applied Intelligence & Robotics

Perception, machine learning, signal decoding, autonomous navigation — the top layer of the stack, built on all layers below it.

Layer philosophy: Intelligence without hardware is an abstraction. What makes these projects interesting is that they are grounded: the BCI classifier runs on signals from real EEG electrodes with real noise; the activity recognition runs on a real wearable sensor; the autonomous car navigates a real track in real time; the robot was built with real motors and real drift that had to be measured and corrected. Machine learning is a tool here, not the point — the point is making things that function in the physical world.

Motor Imagery BCI Classification

64-channel EEG at 250 Hz for left/right hand motor imagery decoding. Three classifiers benchmarked: 1D CNN with channel attention (83.8%), EEGNet (83.4%), CSP+SVM (81.2%). 5-fold stratified CV on BCI Competition 2005 Dataset I.

Respeck Activity Recognition

Chest-worn wearable sensor (12 Hz 3-axis accelerometer). Two-stage hierarchical classifier: Stage-1 coarse (LOCOMOTION/MISC/RUNNING/STATIC), Stage-2 fine within each group (12+ classes). Handcrafted time/frequency features, RF · XGBoost · SVM ensemble. Clinical activity monitoring.
